## 📦 Software Catalog

### 1. S-Player
A high-performance media player designed for multilingual users and content researchers. It features frame-accurate subtitle timing, waveform scrubbing, and robust playback for local or streamed media.

### 2. Speech with Me (AI Speaking Coach)
An interactive language learning app that provides AI-driven conversations and real-time feedback. It utilizes local models via **MLX**, **Ollama**, and **LM Studio**, supporting models like **Qwen2.5-3B** and **Whisper** for maximum privacy.

### 3. PyPilot
A lightweight developer tool for instant execution and management of Python scripts. It supports various environments including **Conda**, **Poetry**, **uv**, and **Hatch** with an integrated internal console.

### 4. Auto Video Downloader (AVD)
A workflow tool for capturing stream sources and automating organized media downloads. Includes a macOS desktop app and a companion Chrome extension.

### 5. Media to Subtitle
A local AI-powered transcription tool that uses **Whisper** models to convert audio and video into text/subtitles (SRT/TXT) without cloud dependency.

### 6. Video Stream Manager
An advanced utility for downloading YouTube videos and playlists, featuring MP3 extraction and multi-language subtitle archiving.

### 7. Clipboard Reader
A lightweight macOS accessibility tool that monitors the clipboard and reads content aloud using natural speech synthesis.
